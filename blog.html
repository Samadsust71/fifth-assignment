What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, allowing programming languages JavaScript to interact with and manipulate HTML documents.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use several built-in methods depending on the type of selection you want to make. Here are some common methods:

  • 1. getElementById() : Selects a single element by its id attribute.
  • 2. getElementsByClassName() : Selects all elements by its class attribute.
  • 3.getElementsByTagName() :Selects the first element that matches a CSS selector.
  • 4. querySelector(): Selects a single element by its id attribute.
  • 5. querySelectorAll(): Selects all elements that match a CSS selector.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element, instead of adding event listeners to individual child elements. This listener can then handle events that are triggered by its child elements as they bubble up through the DOM.

Attaching event listeners to many individual elements can consume memory and affect performance, especially in dynamic or large lists. Event delegation reduces the number of event listeners by utilizing a single listener for multiple elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles in the DOM using JavaScript by accessing and modifying the properties of DOM elements.

You can get, set, or remove attributes of an element using the following methods:

Getting an Attribute: You can use the getAttribute() method to get the value of an attribute.

Setting an Attribute: You can use the setAttribute() method to change or add an attribute to an element.

Removing an Attribute: To remove an attribute from an element, use removeAttribute().

Manipulating Styles: You can manipulate an element’s style by directly accessing the style property of the element, or by modifying the element’s class attributes.
